# Filmorate

## Описание

Сервис, работающий с базой данных фильмов. Позволяет пользователям 
ставить лайки фильмам и получать список популярных фильмов. 

## Технологии и версии

- Java 21
- Spring Boot 3.3.2
- Maven
- Jdbc
- SQL
- Springdoc-openapi

## Установка

Клонировать проект и запустить в любой IDE.

Проект работает с локальной H2 базой данных.

## Документация

Подробное описание эндпойнтов доступно в OpenApi
[документации](https://editor-next.swagger.io/?url=https://raw.githubusercontent.com/A-Cherm/java-filmorate/refs/heads/main/openapi.yaml).

Краткое описание функций сервиса:

- создание, получение и обновление пользователей
- добавление и удаление друзей
- получение общих друзей пользователей
- создание, получение и обновление фильмов
- добавление и удаление лайков к фильмам
- получение жанров фильмов
- получение возрастных рейтингов фильмов
- получение списка популярных фильмов.
